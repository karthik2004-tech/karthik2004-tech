## 🛰️ Featured Project: Satellite Building Detection & Urban Change Analysis

**Internship at NRSC (ISRO)** — National Remote Sensing Centre

Built an end-to-end deep learning pipeline for detecting buildings in satellite imagery and tracking urban expansion over time.

- **Segmentation model**: DeepLabV3+ with a ResNet34 backbone, trained on the **WHU Building Dataset** for pixel-level building footprint detection
- **Change detection**: Extended the work with a **Siamese U-Net** architecture to detect changes in building footprints across time periods, using synthetic change-pair generation for training
- **Urban expansion analysis**: Built a feature to compare multi-year satellite imagery and quantify urban growth, with statistical visualizations
- **Interface**: Wrapped the pipeline in a **Gradio GUI** so the model is usable without touching code
- **Infrastructure**: Trained on Kaggle (P100/T4 GPUs), working within real compute constraints

This project combines semantic segmentation, change detection, and applied geospatial analysis — the kind of full pipeline (data → model → usable tool) that real-world CV work actually requires.
